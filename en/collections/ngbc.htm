Nintendo Game Boy Color

Redesign of the original Game Boy, which became its successor.
Game Boy Color was Nintendo's answer to the fact that the Game Boy was becoming archaic by the end of the 20th century.
The new model offered better parameters, using the same processor, although operating on different clocks, depending on the mode.
Compared to its predecessor, the console requires only two AA cells and the battery life has been extended.
Games dedicated to the "Color" version will not run on the first incarnation of the Game Boy series and its smaller variant called "Pocket".
However, there were titles that would run on an older model, but on Game Boy Color they take advantage of newer hardware, such as Pokémon Gold/Silver.
Of course, the display still has no backlight, so you had to use the console under a light source, or use an attached light accessory.
This unit has a replacement casing, which can be recognized by a smaller speaker grille, flat, matt B and A buttons, as well as a slightly crooked Nintendo logo on a sticker on the back of the unit.
