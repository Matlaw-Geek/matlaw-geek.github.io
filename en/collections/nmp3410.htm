Nokia 3410

Nokia 3330 with a few additions, a better display and a different, less pretty casing, along with a new set of function keys. A successor model to the much more popular 3310.
A phone that... was created. It is not a revolution, but rather an evolution of the older models still being sold at that time.

Significant novelties are the extended navigation and call answering/ending keys, replacing the previously existing OK, C and up and down. Previously, such expanded controls were only used on higher-end models like the 6110, 8210 and 6510.

An interesting fact is that this is the first Nokia phone with support for Java and 3D graphics (!) using OpenGL ES.

This unit, as you can see on the pictures, was heavily used, hence the lack of manufacturer's logo above the screen (which was only a print) and the worn out keys.

Built-in games:

  • Snake II
  • Space Impact (a shoot 'em up game)
  • Bumper (a pinball game)
  • Bantumi (a mancala type game)
  • Link5 (a game looking like go, but played like noughts and crosses)
